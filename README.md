# Java Chapter-Wise Code

A collection of Java programs organized chapter by chapter, covering core concepts from basics to advanced topics. This repository serves as a learning reference and practice resource for Java programming.

## 📖 What's Inside

56 chapters of Java code examples covering topics such as variables, control flow, loops, methods, arrays, object-oriented programming, file handling, generics, multithreading, and more.

## 🚀 Getting Started

1. Clone the repository
   ```bash
   git clone https://github.com/Muhammad-Umer-Randhawa/Java-Chapter-Wise-Code.git
   ```
2. Open any chapter folder and run the `.java` file
   ```bash
   javac FileName.java
   java FileName
   ```

## 🛠️ Prerequisites

- Java JDK 8 or higher
